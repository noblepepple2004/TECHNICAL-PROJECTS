# Case Study: AI Response Evaluation

## Objective

Develop a consistent method for reviewing AI-generated responses.

## Evaluation Criteria

- Accuracy
- Completeness
- Clarity
- Relevance
- Grammar
- Logical reasoning

## Review Process

1. Read the response.
2. Check for factual accuracy.
3. Identify missing information.
4. Improve wording where necessary.
5. Suggest revisions.

## Outcome

Using a structured checklist made evaluations more consistent and easier to compare.
